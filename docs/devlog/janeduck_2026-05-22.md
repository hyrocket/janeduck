### 온보딩 플로우 신규 구현

첫 접속 사용자 대상 3-step 온보딩.

#### 흐름
1. **Step 1** — JaneDuck 인사 ("Hey, I'm JaneDuck!")
2. **Step 2** — 이름 입력 ("What should I call you?") + Skip 가능
3. **Step 3** — 개인화 환영 ("Nice to meet you, [name]!") + 3줄 사용법 안내 → [Start learning 🚀]

#### 온보딩 완료 기억 방식
| 사용자 | 체크 위치 | 저장 |
|---|---|---|
| 로그인 | `app/page.tsx` 서버에서 DB 체크 | `user_profiles.onboarding_done`, `display_name` |
| 게스트 | `/decks` 클라이언트 가드 | `localStorage['jd_onboarding_done']` |

#### 이름 스킵 재요청 로직
- 이름 없이 스킵 시 `onboarding_done = true`이지만 `display_name IS NULL`
- 다음 로그인 시 `app/page.tsx`에서 `display_name IS NULL` → `/onboarding?step=2` 리다이렉트 (Step 2부터, 인사 생략)

#### 테스트 헬퍼
- `/onboarding?reset=1` — localStorage 초기화 후 Step 1부터 재시작

#### 신규 파일
- `app/onboarding/page.tsx` — 온보딩 3-step 클라이언트 페이지
- `app/api/onboarding/complete/route.ts` — POST: display_name DB 저장 + onboarding_done 플래그
- `app/decks/GuestOnboardingGuard.tsx` — 게스트용 클라이언트 가드
- `lib/db/migrations/003_user_profiles.sql` — user_profiles 테이블 신규

#### DB 변경
```sql
CREATE TABLE user_profiles (
  user_id         UUID PRIMARY KEY,
  display_name    TEXT,
  onboarding_done BOOLEAN NOT NULL DEFAULT false,
  created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
```
